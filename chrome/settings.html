Ollama Configuration Required
Ollama does not allow requests from Chrome Extensions by default.

To enable: Add the following to your ~/.zshrc or ~/.bashrc file:
launchctl setenv OLLAMA_ORIGINS "*"

Then restart Ollama for the changes to take effect.

LM Studio Local AI
LM Studio runs AI models locally on your computer with a beautiful UI.

To use:
  1. Download LM Studio from lmstudio.ai
  2. Load a model in LM Studio
  3. Start the local server (usually on port 1234)
  4. Enter the server URL below
Default is http://localhost:1234. Make sure LM Studio server is running.

Osaurus - Native macOS LLM Server
Osaurus runs AI models locally on your Mac with Apple Silicon optimization.

Installation:

Start Osaurus:

Use model name foundation to access Apple's system models (macOS 26+).
