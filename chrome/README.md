# OpenCopilot - Chrome/Zen Browser Version

This folder contains the Chrome and Zen browser compatible version of OpenCopilot.

## Installation

1. Open Chrome or Zen browser
2. Navigate to `chrome://extensions/`
3. Enable "Developer mode" (toggle in top right)
4. Click "Load unpacked"
5. Select this `chrome` folder
6. The extension icon will appear in your toolbar

## Usage

- Click the extension icon to open the sidebar
- Use `Ctrl+Shift+O` (Mac: `Command+Shift+O`) to toggle the sidebar
- Use `Ctrl+Shift+K` (Mac: `Command+Shift+K`) to open settings

## Features

- Native Chrome Side Panel integration
- All features from the main extension
- Optimized for Chrome/Zen browser APIs

## Notes

- Zen browser is Chromium-based and uses the same extension format as Chrome
- This version uses Chrome's `sidePanel` API
- All Chrome extension APIs are supported
